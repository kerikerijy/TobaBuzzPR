## 1. プロジェクト概要

- **プロジェクト名**: TobaBuzzPR
- **目的**: 京都府立鳥羽高等学校の広報を刷新し、中学生へのリーチを強化する
- **対象校**: 京都府立鳥羽高等学校
- **現行HP**: https://www.kyoto-be.ne.jp/toba-hs/ (Movable Type ベース)
- **施策**: Instagram広報の導入 + ホームページの刷新

## 2. 鳥羽高校の強み（コンテンツ素材）

### USP 01：グローバル科の「尖り」
- フランス・台湾との短期交換留学
- 文部科学省「トビタテ！留学JAPAN」→ 府立高校生「海外探Ｑ留学」支援事業の採択実績豊富
- SGH指定校以来受け継ぐ21世紀型探究教育（WWL等）
- グローバル科 前期倍率1.25倍

### USP 02：普通科「ど真ん中」の誠実な強み
- 「特別じゃない」が最大の強み（Trust Equity）
- 挨拶の徹底・落ち着いた校風
- 個々の目標に寄り添う進路指導
- グローバル科との混在環境
- 普通科 中期倍率1.12倍・府内11位

### USP 03：知的ユーモア × 文化の厚み
- カレーうどん探究（尖ったテーマの探究活動）
- 披講研究部（冷泉流歌道）
- 茶道・華道（未生流笹岡）
- レンガ色ブレザー（地域のアイコン）

### 全国レベルの部活動
- 硬式野球部
- 男子バスケットボール部
- ウエイトリフティング部
- その他多数の体育系・文化系クラブ

## 4. 成果物

### Webプレゼン資料
- **パス**: `presentation/index.html`
- **公開URL**: `https://kerikerijy.github.io/TobaBuzzPR/presentation/index.html`
- **内容**: 管理職・職員会議向けの動的プレゼン資料。全4章構成（現状分析、環境分析/ディレクトリ、隠れた資産、戦略提言/ロードマップ）。
- **機能**: 97校ディレクトリのフィルタ・ソート機能、位置情報のインタラクティブ競合マップ（半径5km圏）、レスポンシブデザイン。

### 広報基盤データ
- **パス**: `kyoto_highschool_data.md`
- **内容**: 京都府下全97高校の公式サイトHPおよびInstagram運用状況の全数調査データ（フォロワー数、投稿数、HP連携状況）。

### 入学定員・倍率データ
- **パス**: `kyoto_highschool_admission_ratios.md`
- **内容**: 京都府内の公立高校全校を網羅した「100% Census（全数調査）」データ。専門学科（前期100%募集枠）と普通科（中期募集枠）を対置させ、一本化に向けた「真の需要」を可視化したマスターレポート。
