## 1. 提案の概要

本校の中学生向け広報強化を目的として、**学校公式Instagramアカウントの開設と運用体制の整備**を提案します。

| 項目 | 内容 |
| :--- | :--- |
| **提案内容** | 公式Instagramアカウントの開設・運用ルールの制定 |
| **目的** | 中学生・保護者への認知拡大、第一志望校としての選択率向上 |
| **費用** | 原則無料（Meta Business Suite の標準機能を使用） |
| **運用開始目標** | 2026年○月 |

## 2. なぜ今、Instagramなのか

### 2-1. 府立高校の導入状況（2026年3月調査）

**京都府立高校46校のうち、35校（76.1%）がInstagramを公式運用中**です。

本校（鳥羽高校）は現時点で未開設であり、府立高校の中では少数派に属します。競合する周辺の府立高校（西京・堀川・紫野・桂など）もすでに運用を開始しています。

> Instagram は「見つけてもらうための入口」として、高校広報の標準インフラになっています。未開設は「情報発信の機会損失」と捉えるべき段階です。

### 2-2. 中学生の情報収集行動の変化

- 中学生・保護者が高校を調べる際、**最初に見るのはGoogle検索やSNSであり、学校HPは後半**に確認される傾向があります（各種調査）
- Instagramは「学校の雰囲気・部活の熱量・日常の様子」を一目で伝えられる唯一のメディアです
- 現在の本校HPは更新頻度が低く、リアルタイムの情報発信が困難な構造です

### 2-3. 鳥羽高校が発信すべき強み

本校には発信できる魅力が多数あります。

- **野球部**（甲子園常連クラスの競争力）
- **バスケットボール部・ウエイトリフティング部**（全国レベル）
- **グローバル科**（海外研修・留学プログラム）
- **探究学習・進路実績**
- **立地の希少性**: 京都駅から最も近い公立高校

これらの魅力をリアルタイムに発信する手段が現在ありません。

## 3. 運用体制と管理職の関与

### 3-1. 承認フロー（「管理職承認なしには公開されない」仕組み）

全投稿は**管理職の確認・承認を経てから公開**されます。担当教員が勝手に公開できない設計です。

```
①担当教員が「下書き」を作成
②企画推進部にチャットで連絡
③管理職（教頭等）が内容を確認・承認
④管理職が「公開」ボタンを押す
```

> **担当教員には「下書き作成」の権限のみ付与します。「公開」は管理者権限を持つ方のみです。**
> この仕組みは Meta Business Suite（Facebookが提供する無料の公式管理ツール）で実現します。

### 3-2. 運用責任者と担当者

| 役割 | 担当 |
| :--- | :--- |
| **運用責任者（管理者権限）** | 企画推進部長、教頭 |
| **投稿担当（下書き作成）** | 各部活顧問、企画推進部担当教員 |
| **ガイドライン管理** | 企画推進部 |

## 4. 個人情報・肖像権の取り扱い

### 4-1. 生徒の肖像利用ルール

入学時に保護者から**「SNS・WEB肖像利用同意書」**を回収しています（在校生を含む全学年対象）。

| 同意状況 | 取り扱い |
| :--- | :--- |
| 同意あり | 自然な表情で掲載可 |
| 同意なし・不明 | 顔を隠す加工（スタンプ・ぼかし等）を施す |

> 「同意不明 = 同意なし」の原則を徹底します。

### 4-2. 投稿禁止事項（要点）

- 生徒の合否・成績等に関わる情報
- 個人を特定できる住所等
- 政治的・宗教的内容
- 他校への批判

詳細は「現場運用マニュアル」に明文化済みです。

---

## 5. リスク管理

### 5-1. 炎上・誤投稿への対応

| 事態 | 初動対応 |
| :--- | :--- |
| 誤投稿した場合 | 即座に非表示（アーカイブ）→ 管理者に報告 |
| 炎上・問題視された場合 | 即座に非表示 → 管理職主導で公式見解を決定 |
| 不正ログイン | パスワード変更・二要素認証再設定 |

> **「アーカイブ」は公開を一時停止するだけで、削除ではありません。復元できます。**
> 誤投稿が起きた際も、落ち着いて手順通りに対応できる体制を整えます。

### 5-2. コメント・DM対応方針

- 原則として**コメントへの返信は行いません**（誤解・炎上リスクを回避）
- 問い合わせは「公式HPのお問い合わせフォームへ誘導」するテンプレートを使用します
