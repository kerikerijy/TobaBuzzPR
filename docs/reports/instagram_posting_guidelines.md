# 鳥羽高校公式Instagram運用ガイドライン（案）

## 1. 運用目的
本アカウントは、中学生および保護者をターゲットとし、鳥羽高校のリアルタイムな活動（部活動、行事、日常の探究学習等）を発信することで、学校への理解を深め、第一志望校としての認知（第一想起）を獲得することを目的とする。

## 2. 運用体制
- **運用責任者**: 企画推進部長（または教頭）
- **投稿担当者**: 各部活動の顧問、各年次広報担当教諭、および「生徒広報チーム（仮称）」
- **承認フロー**:
    - 全投稿は公開前に**管理職の承認**を得るものとする。
    - 具体的な技術フローは『Meta Business Suite 技術調査報告書』に基づく「下書き＆レビュー方式」を採用する。

## 3. 投稿内容と基準
- **ハレの投稿**: 学校行事、大会速報、受賞報告、留学プログラム。
- **ケの投稿**: 日常の練習風景、自習室の様子、食堂のメニュー、探究のプロセス。
- **禁止事項**:
    - 特定の個人を過度に称賛または批判する内容。
    - 合否に関わる機密情報。
    - 生徒・教職員のプライバシーを侵害する情報（許可のない顔出し等）。
    - 政治的・宗教的な宣伝活動。

## 4. 肖像権と個人情報の取り扱い
- **同意の取得**: 入学時に「SNS・WEBサイト等への肖像利用に関する同意書（後述）」を全生徒・保護者から取得する。
- **顔出しのルール**: 
    - 「同意あり」の生徒: 自然な表情を公開可能。
    - 「同意なし」または「不明」の生徒: スタンプ、ぼかし、または後ろ姿等で個人が特定できないよう加工する。
- **名前の表記**: 原則として、名字のみ、またはイニシャル表記とする。フルネームの使用は、公式な大会結果等の発表を除き避ける。

## 5. リスク管理と炎上対策
- **コメント・DM対応**: 原則として、コメントへの返信は行わない。重要な問い合わせは公式HPのお問い合わせフォームに誘導する。
- **不適切な書き込みへの対応**: 誹謗中傷や公序良俗に反するコメントが発見された場合、運用責任者の判断により非表示またはブロックの措置を講じる。
- **緊急時の対応**: 投稿内容が問題視（炎上）された場合、速やかに投稿を非表示（アーカイブ）にし、管理職主導で事実確認および公式見解の発表を行う。
