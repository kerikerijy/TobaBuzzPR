# 令和8年度 京都府公立高等学校 志願倍率データ分析

本データは、令和8年度の京都府公立高等学校入学者選抜（前期・中期）の公式広報資料から抽出・算出した志願倍率の分析結果である。
鳥羽高校のInstagram運用戦略を立案する上での重要なエビデンスとして活用する。

## 3. 【特別分析】令和9年度からの入試制度改革（重要）
現在中学2年生が受験する「令和9年度（2027年度）」から、京都府公立高校入試が抜本的に改革される。

### 具体的な変更点
1. **選抜機会の一本化**：現行の「前期」と「中期」を廃止・統合し、新たな「前期（仮称）」の1回のみの入試となる（事実上の一発勝負）。
2. **定員の100%一括募集**：全学校・全学科において、この1回の選抜で定員の100%を募集する。
3. **「独自枠」と「共通枠」の導入**
    *   **独自枠**: 各高校が独自の検査（面接・小論文等）で評価する枠（第1志望のみ出願可）。
    *   **共通枠**: 全校共通の学力検査等で評価する枠（第3志望まで出願可）。
4. **セーフティネットの喪失**：これまでの「前期でチャレンジし、ダメなら中期で手堅く合格を狙う」という受験戦略が通用しなくなる。

### 鳥羽高校広報への影響（緊急性）
現状、鳥羽高校（普通科）は**「前期は高倍率（5.13倍）だが、中期では1.12倍に落ち着く」**という、中期層に支えられた手堅いポジションにある。
しかし、この**「中期（セーフティネット）」が消滅**するため、受験生は「絶対に受かりたい安全校」か「どうしても行きたい憧れの独自枠校」への二極化を迫られる。
鳥羽が安全志向の波に飲まれることなく、**「独自枠（第1志望）で勝負したい憧れの学校」として選ばれるためには、中学生の「第一想起」を勝ち取る日常的なInstagram戦略が猶予なく求められている。**
